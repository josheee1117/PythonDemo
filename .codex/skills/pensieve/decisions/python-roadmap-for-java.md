# 决策：按 Java 开发者背景优化 Python 学习路线

## 背景

用户具备 7 年 Java 开发经验，但当前项目**尚未正式开始学习**。项目中的旧学习进度来自外部 Claude 模板，不作为真实历史。

## 决策

- 不再沿用完整的零基础 17 周路线。
- 保留每日任务、日志、周报、知识库的流程约束。
- 项目文档入口改为 `AGENTS.md` + `LEARNING_SYSTEM.md`，不再以 `CLAUDE.md` 作为主文档。
- 后续学习路线优先聚焦：
  - Pythonic 语法与数据模型
  - 常用标准库
  - 工程化能力
  - FastAPI/Flask Web 后端
  - 数据库与部署

## 直接影响

- Day 1 应从真实起点开始建立记录，而不是沿用模板里的“第 X 天”。
- 后续任务编排要减少基础语法重复训练。
- 默认增加 Java/Python 对照说明。
- 项目练习优先选后端 API、自动化脚本、部署实战。
